# Chicago Corps

> A verified, living directory of mutual aid and community support groups across the Chicago area.

**470 total groups · Verified for 2025/2026 activity · 307+ confirmed active**

🌐 **[lumpencamp.github.io/chicago-corps](https://lumpencamp.github.io/chicago-corps/)**

---

## Features

- **Search & filter** by name, neighborhood, or category
- **Interactive map** with color-coded markers by verification status
- **🙋 Volunteers toggle** — instantly see which groups are accepting volunteers
- **Language tags** — groups serving languages other than English are flagged
- **Urgent need badges** — groups with active donation or supply needs are highlighted
- **➕ Submit a Group** — link in the header to propose additions via GitHub Issues

## Repository Structure

```
chicago_mutual_aid_groups.csv   ← Main data file (edit this to add groups)
build.py                        ← Build script: geocodes, joins data, injects into HTML
index.html                      ← The site (data baked in at build time)
data/
  locations.csv                 ← Street addresses + geocoded coordinates
  contacts.csv                  ← Verified emails and contact forms
  donation_needs.csv            ← Items groups are actively seeking
  urgent_needs.csv              ← Time-sensitive needs flags
  languages.csv                 ← Languages served by each group
  volunteer_needs.csv           ← Volunteer roles and requirements
  coalitions.csv                ← Coalition and partnership relationships
  verification_log.csv          ← History of verification checks
_data/
  api.json                      ← Compiled output (auto-generated by build.py)
  locations.geojson             ← Map points (auto-generated by build.py)
.github/workflows/
  deploy.yml                    ← GitHub Actions: runs build.py on CSV changes
CONTRIBUTING.md                 ← How to add groups, field definitions, category list
```

## Adding or Updating a Group

The easiest way is to [open an issue](https://github.com/lumpencamp/chicago-corps/issues/new).

To edit directly, update `chicago_mutual_aid_groups.csv` and submit a pull request. The GitHub Actions workflow will automatically rebuild and deploy the site. See [CONTRIBUTING.md](./CONTRIBUTING.md) for field definitions and the full category list.

---

## Local Development

No build step needed to browse — `index.html` has all data baked in. To regenerate after editing the CSV:

```bash
python3 build.py
```

Then open `index.html` in a browser.

## Verification Status

| Status | Meaning |
|---|---|
| `VERIFIED` | Confirmed active within ~6 months |
| `LIKELY_ACTIVE` | Strong signals but not fully confirmed |
| `UNCERTAIN` | Mixed signals — needs more research |
| `STALE` | No confirmed activity in 6+ months |
| `INACTIVE` | Confirmed closed |

---

## Deployment

Hosted as a static site on **GitHub Pages** (no Jekyll, no server). The `.nojekyll` file disables Jekyll processing. The `deploy.yml` workflow runs `build.py` automatically whenever `chicago_mutual_aid_groups.csv` or any `data/` file is updated.

To use a custom domain, add a `CNAME` file containing the domain name and configure DNS with your provider.
